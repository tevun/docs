# API

## setup

```text
# tevun setup [<user>]
```

{% hint style="warning" %}
Requires root privilegies
{% endhint %}

## create

```text
$ tevun create <project> <template>
```

## destroy

```text
$ tevun destroy <project>
```

{% hint style="success" %}
Requires user confirmation
{% endhint %}
